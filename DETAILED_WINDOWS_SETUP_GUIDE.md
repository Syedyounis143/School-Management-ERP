## 🔧 Troubleshooting

### Problem 1: "Cannot Extract File"
**Solution:**
- Download 7-Zip: https://www.7-zip.org/
- Right-click file → 7-Zip → Extract Here
- Or try downloading the file again

### Problem 2: "Application Won't Start"
**Solution:**
- Right-click `School Management System.exe`
- Select **"Run as Administrator"**
- If still doesn't work, restart your computer

### Problem 3: "Port 3000 Already in Use"
**Solution:**
- Close other applications using port 3000
- Or change the port in application settings
- Restart the application

### Problem 4: "Windows Defender/Antivirus Blocked It"
**Solution:**
- Click **"More info"** on the security warning
- Click **"Run anyway"**
- The app is safe - it's just a first-time warning

### Problem 5: "Database Connection Error"
**Solution:**
- Check your internet connection
- Ensure you have permission to access the database
- Contact your administrator

### Problem 6: "File Not Found After Extraction"
**Solution:**
- Make sure you extracted the `.gz` file completely
- Check the extraction location
- Try extracting again to a different folder
