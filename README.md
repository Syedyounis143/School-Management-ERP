# 🎓 School Management System (ERP)

A complete **School Management System (ERP)** built using **React, Node.js, and SQLite**.  
This system helps schools manage students, teachers, academics, and reports efficiently.

---

## 🚀 Features

### 👨‍🎓 Student Management
- Add, edit, and view students
- Store student details (name, class, gender, photo)
- Upload student profile image

### 🪪 ID Card Generator
- Generate student ID cards
- Includes **photo + QR code**
- Ready for printing

### 📊 Academics System
- Teacher-based **marks entry**
- Auto-generated **report cards**
- Grade & percentage calculation
- Student-wise report history

### 📥 Bulk Import
- Upload Excel file to add students
- Supports:
  - name
  - class
  - gender

### 💰 Finance (Basic)
- Fee management module
- Extendable for receipts & billing

### 🧠 Admin Panel
- View all students
- View report card data
- Debug and manage system data

### 📅 Attendance System
- Simple attendance marking
- Extendable to biometric

---

## 🛠️ Tech Stack

- **Frontend:** React + TypeScript + Tailwind CSS
- **Backend:** Node.js (Express)
- **Database:** SQLite (lightweight & fast)
- **Other Libraries:**
  - QR Code generator
  - XLSX (Excel import)
  - jsPDF (PDF generation)
